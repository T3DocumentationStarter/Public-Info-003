Wiki
====

Kurzfristiges Ziel für das Wiki unter https://wiki.typo3.org
-------------------------------------------------------------
in Short: Komplettportierung zu https://docs.typo3.org/

- Im ersten Schritt sollte dafür gesorgt werden, dass Wiki nicht zu erweitern. Ergo sollte das wiki auf "Read Only" gestellt werden
- Im zweiten Schritt sollte definiert werden welche Seiten überhaupt noch Relevanz für die Dokumentation haben und welche Seiten auch komplett gelöscht werden können... So können z.B. alte Events etc. gelöscht werden
- Die als relevant gekennzeichneten Seiten werden in RST umgewandelt und auf docs.typo3.org veröffentlicht. Gleichzeitig müssen für die jeweiligen Seiten ein Redirect eingerichtet werden.
- Mit dem Core-Team muss abgesprochen werden, wie die von TYPO3 direkt benötigten Seiten/Einträge im Wiki anderweitig verfügbar gemacht werden (Exeptions/Fehlermeldungen --> more Informations online)
